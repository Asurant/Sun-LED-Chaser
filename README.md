# Sun-LED-Chaser
My first PCB! It's a simple LED chaser made into the shape of a sun. Made this through a tutorial a Hackclub event called Blueprint.  
It works by connecting a 9 volt battery to the header leading to the NE555 and CD4017 combo to blink through each light in a circle.
